## Введение

Данная практическая работа посвящена расчету ключевого параметра производительности сети Ethernet — задержки сигнала (Path Delay Value, PDV). В сетях Ethernet, построенных на разделяемой среде, существует критическое ограничение на время прохождения сигнала между двумя самыми удаленными узлами. Это ограничение, известное как время двойного оборота, необходимо для корректного механизма разрешения коллизий (CSMA/CD).

**Цель работы:** Изучить способы расчета показателей задержки передачи сообщения в сети ETHERNET с учетом ее конфигурации, включающей сегменты различных физических стандартов. Провести расчёт показателя задержки сигнала PDV для предложенных конфигураций сети, сопоставить полученные данные с максимально допустимой величиной (575 bt) и сформировать выводы.

---

## Исходные данные

### Формула расчёта:

PDV = B + L · Kз

где:
- B — база сегмента (битовые интервалы)
- L — длина кабеля (метры)
- Kз — коэффициент задержки среды передачи

**Максимально допустимая величина задержки сигнала PDV: 575 bt**

---

- 10Base-T: 165 м
- 10Base-FB: 1000 м
- 10Base-FL: 970 м

## Расчеты для конфигураций

### Конфигурация 1 (Задание 2)

**Состав сети:**

1. 10Base-2 (200 м) — Левый сегмент
2. 10Base-FL (800 м) — Промежуточный сегмент
3. 10Base-FB (400 м) — Промежуточный сегмент
4. 10Base-FB (800 м) — Промежуточный сегмент
5. 10Base-FB (400 м) — Промежуточный сегмент
6. 10Base-2 (300 м) — Правый сегмент

**Расчеты по формуле PDV = B + L · Kз:**

1. **10Base-2 (левый):** 11,8 + 185 · 0,1026 = 11,8 + 18,981 = **30,781 bt**
   - Примечание: используется L_max = 185 м для 10Base-2

2. **10Base-FL (промежуточный):** 33,5 + 800 · 0,1 = 33,5 + 80 = **113,500 bt**

3. **10Base-FB (промежуточный):** 24 + 400 · 0,1 = 24 + 40 = **64,000 bt**

4. **10Base-FB (промежуточный):** 24 + 800 · 0,1 = 24 + 80 = **104,000 bt**

5. **10Base-FB (промежуточный):** 24 + 400 · 0,1 = 24 + 40 = **64,000 bt**

6. **10Base-2 (правый):** 169,5 + 185 · 0,1026 = 169,5 + 18,981 = **188,481 bt**
   - Примечание: используется L_max = 185 м для 10Base-2

**PDV = 30,781 + 113,500 + 64,000 + 104,000 + 64,000 + 188,481 = 564,762 bt**

**564,762 bt < 575 → ✓ УДОВЛЕТВОРЯЕТ ТРЕБОВАНИЯМ**
